# 於 [!DNL Assets Essentials] 中編輯影片 {#edit-videos}

使用嵌入式 [!DNL Adobe Express] 影片快速動作，讓 Assets 使用者輕鬆建立影片內容的變化版本。 由 [!DNL Adobe Express] 支援之 [!DNL Assets Essentials] 中的快速動作，能提供方便使用的影片編輯選項，包括裁切影片、縮放影片尺寸、剪輯影片以及將影片轉換成 GIF。

若要編輯影片，請前往影片的詳細資料，然後按一下「[!UICONTROL 編輯影片]」。 或者，選取資產並按一下詳細資料，然後按一下右側窗格中可以找到的![剪刀](assets/do-not-localize/cut.svg)圖示。 影片編輯完成後，您可以將新影片儲存為新版本或新資產。

了解更多關於[預覽介面](/help/using/navigate-view.md#preview-assets)的資訊。

## 先決條件 {#prerequisites}

有權存取 [!DNL Adobe Express] 以及 AEM Assets 中至少一個環境。 該環境可以是 [!DNL Assets as a Cloud Service] 或 [!DNL Assets Essentials] 中的任何存放庫。

## 使用 Adobe Express 編輯影片 {#edit-video-using-express}

>[!CONTEXTUALHELP]

>abstract="Experience Manager Assets 現在包含簡單直觀的影片編輯工具 (由 Adobe Express 提供支援)，可增加內容重複使用性並加快內容流通速度。 此編輯選項包括修剪、裁切、調整影片大小以及將 MP4 轉換為 GIF 檔案。"

使用嵌入式 [!DNL Adobe Express] 快速動作可輕鬆將影片轉換成理想尺寸和顯示方向。

### 裁切影片 {#crop-video-using-express}

您可以使用嵌入式 [!DNL Adobe Express] 快速動作，將不需要的部分從影片中移除。 若要使用此功能，請執行下列步驟：

1. 選取影片並按一下「**[!UICONTROL 編輯]**」。
2. 於左側窗格的可用快速動作中，按一下「**[!UICONTROL 裁切影片]**」。
3. 拖曳影片角落的控制點，達到您想要的裁切效果；或根據需要選擇現有的畫面尺寸。
4. 您可以選擇將影片設為靜音或取消靜音。
5. 按一下「**[!UICONTROL 套用]**」。
   ![使用 Adob&#x200B;&#x200B;e Express 裁切影片](/help/using/assets/adobe-express-crop-video.png)

   裁切後的影片可供下載。 您可以將編輯後的資產另存為同一資產的新版本，也可以將其另存為新資產。 ![使用 Adobe Express 儲存影片](/help/using/assets/adobe-express-save-video.png)

### 調整影片大小 {#resize-video-using-express}

數位資產管理中的最終影片內容通常需要調整大小，才能發佈到特定管道。 您可以透過 [!DNL Assets Essentials] 輕鬆調整影片大小，使其符合常見社群管道所需的尺寸，或者也可以根據自訂解析度調整大小。 若要使用 [!DNL Assets Essentials] 調整影片大小，請執行下列步驟：

1. 選取影片並按一下「**[!UICONTROL 編輯]**」。
2. 於左側窗格的可用快速動作中，按一下「**[!UICONTROL 調整影片大小]**」。
3. 於「**[!UICONTROL 調整大小]**」下拉清單中選取社交媒體平台適用的維度。 或者，拖曳影片角落上的控制點，建立所要的裁切大小。
4. 如果需要，使用「**[!UICONTROL 影片比例]**」欄位來縮放影片。
5. 您可以選擇將影片設為靜音或取消靜音。
6. 按一下「**[!UICONTROL 套用]**」以套用您的變更。
   ![使用 Adobe Express 調整影片大小](/help/using/assets/adobe-express-resize-video.png)

調整大小後的影片可供下載。 您可以將編輯後的資產另存為同一資產的新版本，也可以將其另存為新資產。

### 剪輯影片 {#trim-video-using-express}

如果您需要使用較長影片中的某個片段，可以使用「剪輯影片」功能來選取並剪輯影片中的特定片段。 執行下列步驟：

1. 選取影片並按一下「**[!UICONTROL 編輯]**」。
2. 於左側窗格的可用快速動作中，按一下「**[!UICONTROL 剪輯影片]**」。
3. 指定影片的開始和結束時間，剪輯影片的特定部分。 或者，拖曳影片角落上的控制點，建立所要的剪輯片段。
4. 從「**[!UICONTROL 尺寸]**」下拉清單中選取合適的維度。
5. 您可以選擇將影片設為靜音或取消靜音。
6. 按一下「**[!UICONTROL 套用]**」以套用您的變更。
   ![使用 Adobe Express 調整影片大小](/help/using/assets/adobe-express-trim-video.png)

剪輯後的影片可供下載。 您可以將編輯後的資產另存為同一資產的新版本，也可以將其另存為新資產。

### 將影片轉換為 GIF {#convert-mp4-to-gif-using-express}

您可以使用 Adobe Express 將 MP4 影片快速轉換為 GIF 格式。 執行以下步驟：

1. 選取影片並按一下「**[!UICONTROL 編輯]**」。
2. 於左側窗格的可用快速動作中，按一下「**[!UICONTROL 轉換為 GIF]**」。
3. 根據所需的品質選取適當的檔案大小。 此外，選擇橫式、直式或正方形的顯示方向。
4. 拖曳影像角落上的控點，建立所要的影片大小。
5. 按一下「**[!UICONTROL 套用]**」。

   ![使用 Adobe Express 將影片轉換為 GIF](/help/using/assets/adobe-express-convert-video-to-gif.png)

您的影片可以下載為 GIF 格式。 您可以將編輯後的資產另存為同一資產的新版本，也可以將其另存為新資產。

## 限制 {#limitations-video-adobe-express}

* 編輯功能僅支援 MP4 格式的影片。

* 支援的最大來源檔案大小為 1GB。

* 支援的影片任一邊長需大於 46 像素且小於 3840 像素。

* 支援的網頁瀏覽器為 Google Chrome、Firefox、Safari 和 Edge。

* 該功能無法在網頁瀏覽器的無痕模式中開啟。
